# AI Interview Preparation Tool

### Objective

The **AI Interview Preparation Tool** is an innovative platform designed to help job seekers enhance their interview readiness. This tool leverages AI to provide users with a comprehensive suite of features, including resume creation, mock interviews, and personalized feedback, all aimed at improving their chances of success in the job market.

This platform empowers users to create ATS-friendly resumes, simulate real-life interview scenarios, and receive actionable insights for improvement. By combining AI-driven features with an intuitive interface, the tool supports job seekers in building confidence and refining their skills.

---

### Overview

The tool assists users in the following ways:
- **Resume Builder**: Craft professional, tailored resumes optimized for ATS compatibility.  
- **Mock Interviews**: Participate in AI-driven mock interviews and receive real-time feedback to refine responses.  
- **AI Insights**: Analyze interview performance and identify areas for improvement.  

Whether you're a fresher or an experienced professional, this app equips you with the tools needed to excel in interviews and land your dream job.

---

### Features

- **AI-Powered Resume Builder**:  
  Create polished, ATS-friendly resumes customized to job roles.

- **Mock Interview Sessions**:  
  Simulate real-world interviews with AI, practice answering questions, and gain instant feedback.

- **Performance Analytics**:  
  Receive detailed reports on mock interviews to identify strengths and weaknesses.

- **User-Friendly Interface**:  
  Simple and intuitive UI for seamless navigation and a hassle-free experience.

---

### Installation

To get started, follow these steps:

1. Clone the repository:
   ```bash
   git clone https://github.com/vishalbharath/AI-Interview-Preparation-Tool.git
   ```

2. Navigate to the project directory:
   ```bash
   cd AI-Interview-Preparation-Tool
   ```

3. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```

4. Provide your OpenAI API key in the configuration file:
   ```yaml
   api_key: YOUR_OPENAI_API_KEY
   ```

5. Start the application:
   ```bash
   python app.py
   ```

6. Open the app in your browser:
   ```
   http://localhost:8000
   ```

### Usage

1. **Build Your Resume**:
